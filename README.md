# Simmsb Tap

## How do I install these formulae?

`brew install simmsb/tap/<formula>`

Or `brew tap simmsb/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "simmsb/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
